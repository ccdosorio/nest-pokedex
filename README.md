# Ejecutar en desarrollo

1. Clonar repositorio
2. Ejecutar

   ```bash
   npm install
   ```

3. Tener Nest CLI instalado

   ```bash
   npm install @nestjs/cli
   ```

4. Levantar la base de datos

   ```bash
   npm install @nestjs/cli
   ```

5. Stack usado

   - MongoDB
   - Nest

6. Reconstruir la base de datos con la semilla

   ```bash
   http://localhost:3000/api/v2/seed
   ```
